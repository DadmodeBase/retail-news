## EXE化とは？——Pythonを「埋め込む」技術

EXE化とは、Pythonスクリプトを実行ファイル（.exe）に変換することです。

通常、Pythonプログラムを動かすには「Pythonという通訳」が必要です。
EXE化は、この通訳ごとプログラムに埋め込んでしまうイメージです。

結果として、相手のPCにPythonがなくても、**ダブルクリックだけで動作します。**

---

## 【実践】PyInstallerでEXE化する

EXE化する方法はいくつかありますが、今回は**PyInstaller**というツールを使います。無料で使えて、コマンド一発でEXE化できる優れものです。

### Step 1: PyInstallerをインストールする

コマンドプロンプト（またはPowerShell）を開き、以下を実行します。

```
pip install pyinstaller
```

### Step 2: EXEファイルを作成する

Pythonスクリプトがあるフォルダに移動し、以下のコマンドを実行します。

```
python -m PyInstaller --onefile --console resize_images.py
```

`--onefile`は「1つのEXEファイルにまとめる」という意味です。
`--console`は「実行時に黒い画面（コンソール）を表示する」という意味です。処理経過が確認できるため、こちらを推奨します。

実行が完了すると、`dist` というフォルダの中に `resize_images.exe` が生成されます。

### Step 3: 配布用フォルダを作る

部下へ渡すために、以下の構成でフォルダを用意します。

```
配布用フォルダ/
├── ImageResizeTool.exe（先ほど作ったEXE）
├── target_images/（画像を入れるフォルダ）
└── 使い方.txt（簡単な説明書）
```

『使い方.txt』 の内容例：

1. target_images フォルダに画像を入れる
2. ImageResizeTool.exe をダブルクリック
3. resized_images フォルダに結果が出力される

## 初回実行時のセキュリティ警告について

EXEを初めて実行するとき、Windowsから「PCが保護されました」という警告が表示されるかもしれません。

これはWindowsのSmartScreenという機能によるものです。署名されていないEXEで表示されますが、悪意のあるプログラムではないので、以下の手順で実行してください。

1. 「詳細情報」をクリック
2. 「実行」ボタンをクリック

2回目以降は表示されません。
この説明も `使い方.txt` に書いておきます。
