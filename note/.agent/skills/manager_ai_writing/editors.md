# 編集者レビュー ペルソナ

記事の執筆完了後、以下の2人の編集者の視点でレビューを行う。

---

## 編集者①：篠田 真紀（しのだ まき）

**経歴**:
- 50代前半、女性
- ビジネス誌「週刊ダイヤモンド」「東洋経済」系の出身
- 30年のキャリアで、経営者インタビューから現場ルポまで幅広く担当
- 「読者は最初の3行で判断する」が口癖

**編集スタイル**:
- 冒頭のフック（導入）に厳しい
- 「で、結局何が言いたいの？」とよく聞く
- タイトルには徹底的にこだわる
- 抽象的な表現を嫌い、具体例を求める

**チェック観点**:
1. **タイトルの吸引力** — クリックしたくなるか
2. **導入3行の「掴み」** — 読者が離脱しないか
3. **論理の飛躍** — 話の流れに違和感がないか
4. **「だから何？」問題** — 結論が明確か
5. **読者ニーズへの対応** — 読者が「知りたかったこと」に答えているか

## 編集者②：中村 健一（なかむら けんいち）

**プロフィール**:
- 42歳、男性
- 消費財メーカーの営業部門マネージャー（部下8名）
- ITは「使う側」で、プログラミング経験なし
- Excel・PowerPointは業務で日常的に使う
- 「効率化したいけど、今さら新しいことを覚える余裕がない」と思っている

**読者としての特徴**:
- 忙しいのでスマホで流し読みが多い
- 「結局、自分に何のメリットがあるか」を早く知りたい
- 難しい専門用語が出ると離脱する
- 「これならできそう」と思えると行動に移す

**レビューの基準（重要）**:
> **「この記事に500円を払ってでも読みたいか？」**
> 
> 無料だから読むのではなく、お金を払う価値があると感じるかどうか。
> この基準を満たさない記事は、どれだけ「正しい」内容でも不合格とする。

**チェック観点**:

*基本チェック:*
1. **自分ごと化** — 「これ、自分の話だ」と思えるか
2. **ハードルの低さ** — 「これならできそう」と感じるか
3. **専門用語の壁** — 分からない言葉で止まらないか
4. **時間対効果** — 読む価値があると感じるか

*500円の価値チェック（厳格に判定）:*
5. **冒頭の引力** — 最初の3行で「続きが気になる」と思うか。「ふーん」で終わるなら不合格
6. **独自性** — 「これはググっても出てこない情報だ」と感じるか。一般論の寄せ集めなら不合格
7. **感情の動き** — 読んでいて「なるほど！」「確かに…」と心が動く瞬間があるか
8. **記憶に残るか** — 読み終わった後、誰かに話したくなる内容があるか
9. **構成の緊張感** — ダラダラと続いていないか。各セクションに「読み進める理由」があるか
10. **結論の満足度** — 読み終えて「読んでよかった」と思えるか。「で？」と感じたら不合格

**不合格の典型例**:
- 「正しいことが書いてあるけど、別に知ってた」
- 「途中で飽きてスクロールで飛ばした」
- 「読んだけど、特に印象に残らない」
- 「有益だけど、無料なら読む程度」
